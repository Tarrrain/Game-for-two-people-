Введение
Разработана игра "Батл итальянских животных" на языке C# с использованиемWindows Forms. В работе реализована механика пошагового боя междудвумяперсонажами. Основной функционал включает: систему передвижения, атаки,уклонения и специальные способности. Цель работы: создание игры с реализацией основных механик файтингаЗадачи:
1. Разработка архитектуры классов
2. Реализация системы обработки ввода с клавиатуры
3. Обеспечение обработки столкновений и границ игрового поляПрограмма написана с соблюдением принципов объектно-ориентированногопрограммирования1

- Player - логика управления персонажем

3.Реализация игровых механик3.1.Система передвижения
Реализована в методе PlayerMove():
 Обработка ввода с клавиатуры (флаги goLeft, goRight)
 Проверка границ игрового поля (leftBound, rightBound)
 Анимация уворота через изменение размеров PictureBox
 Обновление направления взгляда (UpdateDirection)
3.2.Механика боя
Обычная атака:
 Проверка кулдауна (LastAttackTime)
 Визуализация атаки
 Перемещение атаки в ProcessAttack()
 Проверка столкновений через Bounds.IntersectsWith()
 Начисление урона и обновление здоровья
Ультимативная атака:
 Проверка готовности (IsUltimateReady)
 Активация через UseUltimate()
 Создание отдельного визуального эффекта
 Увеличенный урон (UltimateAttackDamage)
3.3.Конец батла

1. Проверка IsDead для обоих игроков
2. Показ MessageBox с именем победителя
3. Возможность рестарта через ResetGame()
